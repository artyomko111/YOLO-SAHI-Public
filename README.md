# YOLO-SAHI
This repository provides the source code for the thesis project "Improving Small Object Detection from Drones Using SAHI-YOLO".
## Repository Structure
├── code/ # Scripts for training, prediction, evaluation
├── models/ # Custom YOLOv{9..12} .yaml files
├── data/ # Dataset annotations and sample images
├── results/ # Evaluation outputs (mAP, precision, recall)
├── figures/ # Figures from thesis (e.g., failure cases)
├── thesis.pdf # (Optional) Final version of the thesis
└── README.md
